# PySerum

Python client library for interacting with the Project Serum DEX.

## Development

### Setup

1. Install pipenv.

```sh
brew install pipenv
```

2. Install dev dependencies.

```sh
pipenv install --dev
```

3. Activate the pipenv shell.

```sh
pipenv shell
```

### Lint

```sh
make lint
```

### Tests

```sh
make tests
```

### Using Jupyter Notebook

```sh
make notebook
```
